## Vue-Official

Vue - Official (之前是 Volar) 是官方的 VS Code 扩展，提供了 Vue 单文件组件中的 TypeScript 支持，还伴随着一些其他非常棒的特性。

已往,解决TS找不到vue的声明文件的时候使用的插件是Volar(Typescrit Vue Plugin)
